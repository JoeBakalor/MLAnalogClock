## Setup (MacOS)

1. Install nodejs & npm <br />
`brew install node`

2. Install ciaro <br />
`brew install cairo`


3. Intall libjpeg <br />
`brew install libjpeg`


4. Intall pango <br />
`brew install pango`


5. Intall giflib <br />
`brew install giflib`

**Usage**

* Open terminal in /MLAnalogClock, 
`node generateImages.js`
